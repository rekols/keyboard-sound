## keyboard sound

When you press the keyboard will play sound.

### Build dependencies

`sudo apt install g++ qt5-default libqt5x11extras5-dev libxcb-util0-dev libxtst-dev`

### Runtime dependencies

* qt5widgets

* qt5gui

* qt5core

`sudo apt install libqt5core5a libqt5gui5 libqt5widgets5`
